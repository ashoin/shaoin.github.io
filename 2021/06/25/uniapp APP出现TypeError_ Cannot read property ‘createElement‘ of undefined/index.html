uniapp APP出现TypeError: Cannot read property ‘createElement‘ of undefined

发表于2021-06-25|更新于2023-01-12
|阅读量:

问题描述

uniapp H5代码打包和运行都是没有问题的。打包成安卓app也是基本上没有问题的，当要用到第三方js时，会报一些undefined的相关错误

例如：我用 npm 引入阿里对象存储的 ali-oss插件会报 TypeError: Cannot read property 'createElement' of undefined

原因是：uniapp打包APP时没有DOM概念的，ali-oss使用到了DOM，所以会报错。

解决问题

查阅文档与资料，自己封装了一个单图片上传与视频上传并返回存储地址的js文件。

第一步：辅助函数；

第二步：暴露出的图片上传函数；

第三步：文件上传核心函数；

第四步：暴露出的视频上传函数；

文件下载地址：CSDN 0积分下载 或者 uniapp插件市场下载

相关说明

参数 类型 说明
file String 要存放的文件夹名；例如 …/imageList/
filename Blob 被选中的图片或者视频文件名
type String image 或者 video
path Blob 同filename被选中的本地图片或视频文件
stroeAs String 自定义文件名；例如 …/imageList/xxxx.png
